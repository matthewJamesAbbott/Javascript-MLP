# Javascript-MLP

Welcome to the **Javascript-MLP** repository! This project is a Multi-Layer Perceptron (MLP) implementation using JavaScript and HTML. The goal of this repository is to provide a simple and interactive way to understand the foundations of neural networks.

## Table of Contents
- [About the Project](#about-the-project)
- [Features](#features)
- [Getting Started](#getting-started)

## About the Project

This project aims to demonstrate the workings of a Multi-Layer Perceptron (MLP), a fundamental building block of modern neural networks. It is designed to be lightweight and accessible, making it an excellent resource for beginners in machine learning.

### Built With
- **HTML**: For the user interface.
- **JavaScript**: For the implementation of MLP algorithms.

## Features

### 🧠 **Complete Neural Network Suite**
- **Flexible Architecture**: Configure input/hidden/output layers with multiple activation functions
- **Advanced Training**: Batch processing, progress tracking, and configurable epochs
- **Comprehensive Evaluation**: K-fold cross-validation, precision/recall/F1-score metrics
- **Model Persistence**: Save/load trained models as JSON files
- **Real-time Prediction**: Interactive prediction with confidence scores

### 🎯 **Key Capabilities**
- Multi-layer perceptron with deep network support
- Multiple activation functions (Sigmoid, Tanh, ReLU, Softmax)
- CSV data loading and synthetic data generation
- Detailed performance analytics and visualization
- Web-based interface requiring no installation

### 💡 **Perfect For**
- Learning neural network fundamentals
- Research and prototyping
- Educational demonstrations
- Quick machine learning experiments

## Getting Started

To get a local copy up and running, follow these simple steps:

### Prerequisites
Make sure you have a modern web browser installed.

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/matthewJamesAbbott/Javascript-MLP.git
   ```
2. Open the `MLP.html` file in your web browser.

### Quick Demo
1. **Create Network**: Click "Create Network" with default settings
2. **Generate Data**: Click "Generate Test Data" for sample dataset
3. **Train Model**: Click "Train with Progress" to see real-time training
4. **Test Prediction**: Click "Predict" to see classification results
5. **Evaluate**: Click "Run All Evaluation Metrics" for performance analysis

For a complete guide, see [TUTORIAL.md](TUTORIAL.md)
